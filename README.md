# 可视化内容选择
1. 城市分布 & 能级对比
    + 内容：9 家大厂在不同城市的岗位数量分布，按一线 / 新一线 / 二线城市统计。
    + 图形：
        - 中国地图 + 气泡 / 热力图（岗位数）
        - 条形图：城市岗位 Top10，对比腾讯/字节/阿里等占比
2. 薪资区间分布（按公司对比）
    + 内容：各大厂的招聘薪资区间分布，比如 10-15k、15-20k、20-30k…
    + 图形：
        - 分箱柱状图 / 堆叠柱状图（每家大厂一组）
        - 箱线图：对比 9 家的 中位数、上下四分位、极值
3. 学历要求对比（本科 / 硕士 / 博士 / 大专）
    + 内容：每个公司对不同学历的偏好：本科比例多少、硕士比例多少等。
    + 图形：
        - 堆叠条形图（公司在 x 轴，学历比例在 y 轴）
        - 雷达图：每家公司一个雷达，维度是不同学历比例
4. 岗位类型结构（研发 / 产品 / 运营 / 市场 / 职能）
    + 内容：每家大厂内部，各岗位类型所占比例。
    + 图形：
        - 环形图/玫瑰图：单公司岗位结构
        - 分组条形图：9 家大厂的“研发岗位占比”对比

5. 工作经验要求分布（应届 / 1-3 年 / 3-5 年 / 5 年以上） 
    + 内容：每个公司更偏好什么经验段？应届生岗位比例有多大？
    + 图形：
        - 堆叠条形图：公司 vs 经验段比例
        - 雷达图：公司对不同经验层级的需求强度
6. “加班强度”/工作强度标签可视化 
    + （需要你在爬数据时判断字段，比如是否有“加班”“大小周”“弹性工作”等关键字）
    + 内容：含“加班”“大小周”“高强度”等关键词的岗位比例，分公司统计“高强度”岗位比例
    + 图形：
        - 条形图：每家公司的“疑似 996 岗位占比”
        - 词云：JD 描述里的“高频用语”，看是不是“弹性、奋斗、快节奏、压力大”之类
7. 校招 vs 社招 岗位比例 
    + 内容：区分招聘类型（校招 / 社招 / 实习），对比各大厂对学生群体的友好度。
    + 图形：
        - 堆叠条形图：公司 vs 校招/社招/实习占比
        - 时间轴（如果有时间数据）：不同月份校招岗位变化情况
8. 技术栈 / 技能关键词可视化（面向 CS 专业最有感觉） 
    + 内容：对技术岗 JD 做分词，统计出现频率最高的技能：
        - Java / C++ / Python / Go
        - 大数据 / 算法 / 推荐 / 分布式 / 微服务 / 云原生等
    + 图形：
        - 词云：每家公司各自一张“技术栈词云”
        - 雷达图 / 堆叠条形：对比各家公司对某几项关键技术的需求强度
9. 薪资与学历/经验的关系（“门槛 & 天花板”）
    + 内容：本科 vs 硕士 在同一公司的平均薪资对比,1-3 年 vs 3-5 年 vs 5+ 年的薪资提升幅度
    + 图形：
        - 分组条形图：同公司，不同学历的平均薪资
        - 折线图：随着年限提升的平均薪资曲线，各公司一条线
10. 城市生活成本 vs 实际薪资（“性价比城市”） 
    + （生活成本可以用城市等级 / 租房均价等公开数据粗略代替）
    + 内容：
        - 用“平均月薪 / 参考房租”简单构造一个“打工性价比指数”
        - 对比在不同城市去同一家公司，哪儿更“值”？
    + 图形：
        - 散点图：x=生活成本（或租金），y=平均薪资；斜率体现性价比
        - 排名条形图：大厂工作“最划算城市 Top10”
11. 福利标签 / 企业文化关键词可视化 
    + 内容：从福利和描述字段中提取关键词：
        - 五险一金 / 年终奖 / 期权 / 免费三餐 / 房补 / 弹性打卡 / 带薪病假等
        - 看哪家更喜欢写“福利好话术”，哪家更“直男硬核”
    + 图形：
        - 词云：福利关键词
        - 堆叠条形：公司 vs 某类福利出现比例（比如写到“期权、股票”的岗位占比）。
